# Paintbot

CNC Pigment

## Cloning the Repository

To clone this repository to your machine, you can use either HTTPS or SSH:

### Using HTTPS
```bash
git clone https://github.com/ericubey/paintbot.git
cd paintbot
```

### Using SSH
```bash
git clone git@github.com:ericubey/paintbot.git
cd paintbot
```

After cloning, you'll have a local copy of the paintbot repository on your machine.
